# Daily Habit Tracker

A simple Flask-based habit management system.

## Features

- Add daily habits
- Mark habits complete for today
- Track current streaks automatically
- Show completion percentage for today
- Store data in `habits.json`

## Run locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the app:
   ```bash
   python app.py
   ```
3. Open `http://127.0.0.1:5000` in your browser.
